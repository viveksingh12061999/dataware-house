## Source Data Generation Framework

The project includes a reusable Python framework responsible for generating source data that simulates enterprise systems.

The framework consists of:

- Configuration module
- Business constants
- Shared utility functions
- Master generator
- Logging infrastructure

All source datasets generated by the framework comply with the Source Data Contract defined for the project.

## Logging

The project uses Python's logging module for centralized logging.

Features:
- Console logging
- File logging
- Timestamped messages
- Exception logging

All ETL components use the same logging configuration.
